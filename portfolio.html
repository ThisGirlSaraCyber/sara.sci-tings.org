Sci-Tings

An open-source, collaborative data science laboratory aimed at expanding access to data and information built with AWS.

Go to sci-tings.org

Architectural Diagram



View in new window

This project was built using the following AWS tools:

  • S3
  • Route53
  • CloudFront
  • CodePipeline
  • CodeBuild
  • Lambda

The goal of this project is to learn web development (HTML/CSS/JavaScript), deploying websites using AWS, and data science in R and Python. The goal of Sci-Tings is to create a virtual data science laboratory for users to share, create, and discuss their workflows in a centralized place.
